## Функции
 - get_mask_card_number (Функция выдает макску карты в формате: XXXX XX** **** XXXX)
 - get_mask_account (Функция выдает макску счета в формате: **XXXX)
 - mask_account_card (Функция выдает счет или карту с готовой маской)
 - get_date (Функция форматирует дату и время в формат: ДД.ММ.ГГГГ)
 - filter_by_state (Функция сортирует данные по ключу операции 'state')
 - sort_by_date (Функция сортирует данные по дате)
 - filter_by_currency (Функция принимает на вход список словарей и возвращать итератор, который поочередно выдает транзакции, где валюта операции соответствует заданной (например, USD))
 - transaction_descriptions (Функция принимает список словарей с транзакциями и возвращает описание каждой операции по очереди)
 - card_number_generator (Функция выдает номера банковских карт в формате XXXX XXXX XXXX XXXX, где X — цифра номера карты)
